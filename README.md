## ⚙️ Features

- **Tag all group members** with the `.tagall` command
- **Admin restricted usage** (only group admins can use certain commands)
- **Games** like Tic-Tac-Toe for interactive group engagement
- **Text-to-Speech** with `.tts`
- **Anti-link and Anti-sticker detection** for group safety
- **Warn and manage group members** with admin control

### 🎨 Sticker & Media Tools

- **`.sticker`** — converts a tagged image into a high-resolution sticker
- **`.stickerv`** — converts a tagged video into an HD video sticker
- **`.toaudio`** — converts tagged video/document audio (mp4, docs, tagged videos) into a standalone audio file

### 🛡️ Advanced Antidelete / Antiviewonce Suite

- **Antidelete** — recovers messages deleted from a chat and reposts them
- **Antiviewonce** — automatically resends any view-once media received in any chat back to your own private DM, so it's never lost even if the sender's view-once restriction would otherwise prevent a second look
- **Anti-edit** — detects and reports when a message has been edited after sending
- **Anti-status-delete** — recovers and reposts statuses that were deleted before you could view them
- **Status auto-save on reply** — replying to a contact's status saves that status media for you automatically

### 🖧 Mirrored Bot Network (Multi-Session)

- **`.megabot`** — launches additional bot sessions under the same server via a Master Bot controller
- Multiple users can pair their own WhatsApp number to the same server (RAM permitting), each running as an independent minibot session
- All paired minibots share equal command privileges with one exception: only the Master Bot can add further bots to the server — this keeps server resource allocation under a single point of control

## 📖 About

Mega WhatsApp Bot assists group admins by providing them with tools to efficiently manage large WhatsApp groups, plus a multi-session architecture that lets several users share one hosted bot server. The bot uses the Baileys library (via the `@lordmega/baileys` fork) to interact with the WhatsApp Web API and supports multi-device features.

It is lightweight and can be easily customized to add more commands as per your requirements. The bot runs in a Node.js environment and provides QR code or pair-code based authentication to link your WhatsApp account.

This bot uses [`@lordmega/baileys`](https://www.npmjs.com/package/@lordmega/baileys) — a maintained fork of Baileys tailored for MegaBot MD's multi-session architecture.

## 🛠️ Setup & Installation

### Prerequisites

- Node.js installed on your system
- Git installed (for cloning the repository)

### Step-by-Step Setup

1. **Clone the repository:**

    ```bash
    git clone https://github.com/LORDMEGA-MD/MEGABOT5.git
    cd MEGABOT5
    ```

2. **Install the dependencies:**

    ```bash
    npm install
    ```

3. **Run the bot:**

    ```bash
    node index.js
    ```

4. **Scan the QR code / enter pair code:**

    Once the bot starts, a QR code (or pair code prompt) will appear in the terminal. Use the Linked Devices feature in WhatsApp to connect your WhatsApp account with the bot.
